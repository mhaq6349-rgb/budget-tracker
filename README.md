# Budget Tracker

A single-file personal budget tracker. Download and open — no server needed.

## Features

- **Income / Expense tracking** with category selection
- **SVG donut chart** showing expense breakdown
- **Net balance** with income vs expense summary cards
- **Filter** transactions by All / Income / Expense
- **localStorage** persistence — data stays between sessions
- **Works offline** — open `index.html` directly from `file://`

## Usage

1. Download `index.html`
2. Open in any browser
3. Start adding transactions

Or serve with any static server:
```bash
py -m http.server 8000
```

## Categories

| Income | Expense |
|--------|---------|
| Salary, Freelance, Investments, Gifts, Refunds | Food & Dining, Transportation, Shopping, Entertainment, Bills & Utilities, Health, Education, Groceries, Rent |
